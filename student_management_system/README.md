# Student Management System

**Organization:** WeIntern Pvt Ltd  
**Task:** Week 1 - Task 1

---

## Objective
Console-based Student Management System with menu-driven interface using Python OOP.

---

## Features
- Add new student
- View all students
- Search by ID or Name
- Update student details
- Delete student (with confirmation)

---

## Technologies
- Python 3.x
- OOP (Classes & Objects)
- File Handling (JSON)
- Exception Handling

---

## Folder Structure
```
student_management_system/
├── main.py        # Menu & main logic
├── models.py      # Student class
├── utils.py       # StudentManager class
└── README.md
```

---

## How to Run
```bash
cd student_management_system
python main.py
```
